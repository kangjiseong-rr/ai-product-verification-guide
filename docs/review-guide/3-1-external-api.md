# 3.1 외부 모델 API 연동

## 확인 질문

**외부 모델, 예를 들어 OpenAI 등과 연동 시 호출 로그나 API Key가 존재하는가?**

## 항목

[항목 3.1 API 연동]

## 서류 확인 사항

### STEP 1

신청서에 **외부 모델 연동 여부**를 체크했는지 확인합니다.

다음과 같은 알려진 AI 서비스를 사용하는지 확인합니다.

- 오픈AI(OpenAI)
- 앤트로픽(Anthropic)
- 구글(Google)
- 네이버클라우드
- LG AI연구원 등

## 현장 확인 사항

### STEP 1

호출 로그나 API Key 등 연동 시 남는 흔적이 존재하는지 확인합니다.

- AI 기능 실행 시 호출 로그에 호출 내역이 추가되는지 확인합니다.
- API Key, 호출 시각, 호출 대상 모델, 요청/응답 로그 등 연동 흔적을 확인합니다.

## 판정 기준

아래에 해당되면 **PASS**로 판정합니다.

1. 알려진 AI 서비스를 활용하고 호출 로그나 API Key 등이 확인 가능함

!!! warning "공개 문서 작성 시 주의"
    실제 API Key, 토큰, Authorization 헤더, 내부 서버 주소는 문서에 그대로 노출하지 않습니다. 예시가 필요한 경우 `sk-****`, `Bearer ****` 등으로 반드시 마스킹합니다.
