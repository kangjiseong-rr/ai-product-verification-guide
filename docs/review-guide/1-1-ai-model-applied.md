# 1.1 인공지능 적용

## 확인 질문

**머신러닝, 딥러닝, LLM 등 인공지능 모델이 적용되어 있는가?**

## 항목

[항목 1.1 인공지능 적용]

## 서류 확인 사항

### STEP 1

기업이 신청서에 기재한 **인공지능 기능 명세**를 확인합니다.

### STEP 2

신청서의 기능별 **AI 구현 명세**를 확인하여 사용된 인공지능 모델을 목록화합니다.

### STEP 3

사용된 모델이 **인정 인공지능 모델**인지 확인합니다.

!!! note
    인정 인공지능 모델 목록은 [인정 인공지능 모델](../04-recognized-ai-models.md)을 참조합니다.

## 목록화 예시

| 기능명 | AI 구현 방식 | 모델 | 비고 |
|---|---|---|---|
| 챗봇 응답 생성 | 파인 튜닝 | Qwen3.5-27B | 한국어 특화 튜닝 |
| 판매량 예측 | 자체 학습 모델 | 사이킷런(scikit-learn)의 GLM 모듈 | - |

## 판정 기준

아래 중 하나 이상이 해당되면 **PASS**로 판정합니다.

1. 인정 인공지능 모델을 사용한 경우
2. 항목 1.2에서 학습 및 추론 특성이 있는 모델을 사용하였음을 증빙한 경우

## 유의사항

인정 인공지능 모델을 사용하지 않은 경우, 점검 항목 **1.2 학습·추론 특성 존재**에서 추가 판정합니다.
