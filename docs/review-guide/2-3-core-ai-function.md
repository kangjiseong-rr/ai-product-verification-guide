# 2.3 핵심 AI 기능 확인

## 확인 질문

**인공지능 모델이 적용된 핵심 기능이 정상적으로 동작하는가?**

## 항목

[항목 2.3 핵심 AI기능 확인]

## 현장 확인 사항

### STEP 1

정상 입력 데이터 범위 내에서 AI 기능이 정상 수행되는지 확인합니다.

- 영역
- 구간
- 조건

또한 비정상 입력 데이터에 대해서는 예외 처리하는지 확인합니다.

### STEP 2

학습 기능이 있을 경우, 학습 데이터 추가 및 업데이트 후 다음 사항을 확인합니다.

- 기존 기능 동작 여부
- 신규 데이터가 반영되는지 여부

## 판정 기준

아래에 모두 해당되면 **PASS**로 판정합니다.

1. AI 기능이 정상적으로 동작하는 경우
2. 학습 기능이 있을 경우, 학습으로 인해 출력값이 변경됨이 확인된 경우

## 유의사항

- 데이터 입력 범위는 사용자 매뉴얼, 제품 설명서, 신청서 등을 통해 확인할 수 있습니다.
- 기능 확인을 위해 경계값 분석, 동등 클래스 분할 등 다양한 테스트 기법들을 적용할 수 있습니다.
