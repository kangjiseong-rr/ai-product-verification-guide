# 2.2 생산성 향상

## 확인 질문

**인공지능 모델이 해당 제품의 성능 향상이나 편의성에 기여하는가?**

## 항목

[항목 2.2 생산성 향상]

## 서류 확인 사항

### STEP 1

기업이 신청서에 기재한 **AI 적용 목적**이 아래 유형에 속하는지 확인합니다.

| 유형 | 확인 내용 |
|---|---|
| 기능 추가 | AI 적용을 통해 기존 제품에 없던 핵심 기능이 추가된 경우 |
| 기능 개선 | AI 적용을 통해 출력의 정확도 및 품질을 개선시킨 경우 |
| 사용자 편의성 | AI 적용을 통해 핵심 기능에서의 사용자 입력·조작 단계 단축 및 반복작업을 감소시킨 경우 |
| 접근성 개선 | AI 적용을 통해 텍스트, 음성, 이미지 등 다양한 입력 방식을 지원하는 경우 |
| 자원 효율화 | AI 적용을 통해 CPU/GPU 사용량, 응답 시간, 처리량 등을 개선한 경우 |

## 현장 확인 사항

### STEP 1

AI 적용 목적에 따라 아래 사항을 확인합니다.

| AI 적용 목적 | 현장 확인 사항 |
|---|---|
| 기능 추가 | 확인 사항 없음 |
| 기능 개선 | 확인 사항 없음 |
| 사용자 편의성 | 개선 결과에 대한 증빙 확인 |
| 접근성 개선 | 개선 결과에 대한 증빙 확인 |
| 자원 효율화 | 개선 결과에 대한 증빙 확인. 예: GPU 사용량 비교표 |

## 판정 기준

아래에 모두 해당되면 **PASS**로 판정합니다.

1. AI 적용 목적이 기능 확장, 기능 고도화 등 5개 유형에 해당됨
2. 현장 확인을 통해 증빙이 확인됨

## 유의사항

기능 확장, 기능 고도화 등 5개 유형은 필요시 전문가 자문을 통해 생산성 향상을 검토하고 추가될 수 있습니다.
