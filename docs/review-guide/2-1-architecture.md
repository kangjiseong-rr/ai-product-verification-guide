# 2.1 구조 확인

## 확인 질문

**아키텍처 설계도 상에 인공지능 모델의 위치가 명확하며, 제품 내부 로직과 필수불가결하게 결합되어 동작하는 구조인가?**

## 항목

[항목 2.1 구조]

## 서류 확인 사항

### STEP 1

기업이 제출한 **아키텍처 설계도** 상에 인공지능 모델 컴포넌트 위치가 명확히 표시되어 있는지 확인합니다.

확인할 내용은 다음과 같습니다.

- AI 모델이 구동되는 운영 환경이 식별되어 있는지 확인합니다.
    - 서버
    - Cloud
    - 외부 AI API
    - 사용자 단말 등
- 아키텍처 상에서 모델 위치가 명확히 표현되었는지 확인합니다.

### STEP 2

AI 모델이 제거되었을 때 제품 구동에 영향을 받는지를 확인합니다.

- AI 모델 컴포넌트를 삭제할 경우, 제품 기능 수행이 불가능한지 확인합니다.
- 또는 성능이 저하되는지 확인합니다.

## 판정 기준

아래에 모두 해당되면 **PASS**로 판정합니다.

1. AI 모델 위치가 명확히 표시됨
2. AI 모델 컴포넌트를 삭제할 경우, 핵심 기능 수행이 불가능하거나 성능이 저하됨이 확인됨
