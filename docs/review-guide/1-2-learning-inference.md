# 1.2 학습·추론 특성 존재

## 확인 질문

**모델에 학습 또는 추론 등 인공지능 특성이 존재하는가?**

## 항목

[항목 1.2 학습·추론 특성 존재]

## 서류 확인 사항

### STEP 1

항목 1.1 점검 결과, **인정 인공지능 모델**을 사용한 경우 자동 **PASS** 처리합니다.

### STEP 2

인정 목록에 없는 모델을 사용한 경우, **현장 확인**이 필요합니다.

## 현장 확인 사항

### STEP 1

시연을 통해 학습 및 추론 특성을 확인합니다.

확인 예시는 다음과 같습니다.

- 학습 데이터 또는 모델 변경 시 출력값, 즉 추론 결과가 달라질 수 있음을 확인합니다.
- 모델 학습, 정확도 검증 등 인공지능 모델 개발 과정을 시연으로 확인합니다.
- 학습되지 않은 데이터에 대한 출력값, 즉 추론 결과가 생성됨을 시연으로 확인합니다.

## 판정 기준

아래 중 하나 이상이 해당되면 **PASS**로 판정합니다.

1. 인정 인공지능 모델을 사용함
2. 시연을 통해 학습 및 추론 특성이 확인됨

## 유의사항

- 샘플 데이터셋, 시나리오 등 기업에게 시연 준비를 요청해야 합니다.
- GPU, NPU 사용량을 모니터링하여 실제 학습 연산이 일어나는지 여부를 추가로 확인할 수 있습니다.
