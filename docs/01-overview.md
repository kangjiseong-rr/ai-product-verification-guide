# 제도 소개

## 목적

AI 제품 및 서비스 확인제는 **인공지능 기술 적용이 확인된 제품 및 서비스가 조달 시 우선 고려**되도록 하여, 인공지능 기술 도입을 촉진하고 활용을 확산하기 위한 제도입니다.

## 법적 근거

- 「인공지능 발전과 신뢰 기반 조성 등에 관한 기본법」 제16조 제3항

## 적용 대상

- 인공지능 또는 인공지능기술을 활용한 소프트웨어
- 인공지능 또는 인공지능기술을 활용한 기기 또는 장치
- 인공지능 또는 인공지능기술을 활용한 클라우드 기반 서비스, 즉 SaaS

## 개요

**AI 제품 및 서비스 확인제**는 공공조달 시 우선 고려대상이 되는 AI 제품 및 서비스를 확인하기 위한 제도입니다.

## 확인 영역

확인 기준은 크게 다음 세 영역으로 구성됩니다.

| 구분 | 주요 확인 내용 |
|---|---|
| 인공지능 모델 확인 | 제품·서비스에 인공지능처리 연산체계가 적용되어 있는지, 학습·추론 특성이 있는지, 입력 데이터가 모델을 거쳐 유의미한 결과를 출력하는지 확인합니다. |
| 인공지능 기능 확인 | AI 모델 또는 AI 기술이 제품 구조 안에서 명확한 위치를 가지며, 제품 구동 체계와 필수불가결하게 결합되어 정상 동작하는지 확인합니다. |
| 외부 모델 연동 확인 | 외부 AI 모델 또는 API와 연동된 경우 호출 로그나 API Key 등 연동 흔적을 확인할 수 있는지 확인합니다. |
